CSS

  • Selector defines the element or attributes to which the declarations will apply
  • Declarations include the CSS property we want to apply and the value property
  • Margin adds space around the outside of an element, acting as a buffer
  • Padding adds space around the inside of an element,acts similar as a margin except it increases size of an element
  • Combining two selectors and seperating them with a comma allows declarations to affect both or more selectors.Concise, faster code

  • SSH keys are needed so you dont have to put in your password everytime you make a change to a repository
  • A repository is the common name for what your code is called in github
  • git status: checks what branch we are currently on

  • A pull request will merge whatever changes you made to the current branch
  • git add -A: commits all files in the working branch
  • git commit -m "comment here": adds comment to commit for the branch
